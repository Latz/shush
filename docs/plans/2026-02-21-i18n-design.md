# i18n Design — Shush! Extension

## Goal
Add internationalisation to all user-facing strings using Chrome's native `_locales` API.

## Locales
`en` (base), `de`, `fr`, `es`, `ja`, `zh_CN`

## Approach
Chrome MV3 native i18n:
- `_locales/{locale}/messages.json` per language
- `__MSG_key__` substitution in `manifest.json`
- `chrome.i18n.getMessage('key')` in JS files
- `popup.html` heading (`🤫 Shush!`) stays hardcoded — brand element

## Branded strings (English-only across all locales)
- Extension name: "Shush!"
- Action title: "Shush!"
- Mute/unmute button labels: "Shush!" / "Unshush!"
