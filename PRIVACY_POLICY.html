🤫 Shush! — Privacy Policy

Last updated: March 2026

Shush! is a Chrome extension that helps you find and mute noisy browser tabs. This policy explains what data the extension accesses and what happens to it.

Data collected

None. Shush! does not collect, store, transmit, or share any personal data or usage information. No analytics, no crash reports, no telemetry of any kind.

What the extension reads locally

To do its job, the extension reads the following information from your browser. This information never leaves your device:

  • Tab titles and URLs — to display tab names in the popup and context menu
  • Tab audio state — to detect which tabs are playing sound
  • Tab mute state — to show correct Shush / Unshush button labels

Script injection (<all_urls> permission)

When you mute a tab, the extension injects a small script into that tab to silence <audio> and <video> elements directly. This is necessary because Chrome's built-in mute API does not reliably silence audio in all Chromium-based browsers.

The injected script sets .muted = true on media elements and nothing else. It reads no page content, accesses no user data, and sends nothing anywhere. Injection happens only when you explicitly click Shush! or Unshush! — never automatically.

Third parties

The extension communicates with no external servers and includes no third-party code or libraries.

Changes to this policy

If this policy changes, the updated version will be published at the same URL with a new "last updated" date.

Contact

Questions can be directed to the developer via the Chrome Web Store listing.
